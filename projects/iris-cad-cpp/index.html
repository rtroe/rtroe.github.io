IrisCAD - Alpha v 0.1.0

Overview

IrisCAD is a Cross-Platform Computer Aided Design Software Suite, which allows Designers too create and analyise 3D parts. IrisCAD's versitlity comes from it's use of OpenCascade as the backend modeling Kernel, coupled with an integrated Python Interperter, allowing for not only high level modeling, but also complete automation of any operation.

Features

Many of the previous modeling functions are being ported over from an earlier version of IrisCAD too the OpenCascade version.

Open and Import Industry Standard File Types

  • IrisCAD++ can work with the following industry standard file formats
    • *.STEP
    • *.STL
    • *.IGES

Integrated Python Interperter

An Integrated Python Interpreter allows

  • Project Automation through Macros
  • Full access too Python's Functions

Cross Platform
